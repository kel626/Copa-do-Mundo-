# 🏆 World Cup Dashboard - Dark Edition

Sistema de gerenciamento de seleções da Copa do Mundo, desenvolvido para consolidar conhecimentos em desenvolvimento web full-stack e UX Design.

## 🎯 Destaques do Projeto
* **Back-end Seguro:** Implementado em PHP utilizando **PDO**, garantindo proteção contra SQL Injection.
* **Lógica de Ranking:** Algoritmo que ordena as seleções automaticamente pelo número de títulos mundiais.
* **UX Dinâmica:** Renderização de estrelas (⭐) baseada nos dados do banco MySQL.

## 🛠️ Tecnologias
* **Linguagem:** PHP 8.2
* **Banco de Dados:** MySQL
* **Estilização:** CSS3 (Custom Properties) & Pico.css
* **Versionamento:** Git
